## Code quality

### Component elements order

1. Pola statyczne
2. inject()
3. Zapytania widoku (@ViewChild, @ViewChildren itp.)
4. Wejścia/Wyjścia (input(), output(), @Input).
5. Stan lokalny (Sygnały i zwykłe zmienne).
6. Sygnały pochodne (computed).
7. Efekty (effect)
8. Konstruktor.
9. Lifecycle hooks (np. ngOnInit).
10. Metody publiczne.
11. Metody chronione (protected).
12. Metody prywatne (private).
